包简介
=======================================

Go语言是使用包来组织源代码的，包（package）是多个 Go 源码的集合，是一种高级的代码复用方案。Go语言中为我们提供了很多内置包，如 fmt、os、io 等。


包导入
-------------------

.. code-block:: go 

    # 单行导入方式
    import "包 1 的路径"


    // 多行导入方式
    import (
        "包 1 的路径"
        "包 2 的路径"
    )

    //全路径导入
    import "lab/test"

    //相对路径导入
    import "../a"
